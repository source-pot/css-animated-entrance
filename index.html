Animated intro on this site

You may have noticed when you arrived here that there's a little welcome message on the website before you get to this main content. It's created in CSS with a tiny sprinkle of javascript to control it.

Here's the CSS involved in this animated intro and here's the Javascript.

It's neither big nor clever, but it entertained me to create it. To trigger the animation, a bit of Javascript is required to remove the "preview mode" class and add the main class: document.body.classList.replace('page--preview-active', 'page')
